# Food Delivery Backend API

A RESTful backend API for a Food Delivery Application built using Node.js, Express.js, MongoDB, and JWT Authentication.

## Features

* User Registration and Login
* JWT Authentication
* Role-Based Authorization (Admin, Client, Vendor, Driver)
* Restaurant Management
* Food Management
* Category Management
* Order Management
* Password Reset Functionality
* Protected Routes
* Logout Functionality
* MongoDB Database Integration

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* bcryptjs

## API Modules

* Authentication APIs
* User APIs
* Restaurant APIs
* Food APIs
* Category APIs
* Order APIs

## Installation

```bash
npm install
npm run server
```

Create a `.env` file and add required environment variables before running the project.
